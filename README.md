# Spot Shell (spotsh)
Utility for launching &amp; connecting to a shell on a spot EC2 instance

## Building

```bash
make
make test
```

## Installing

Copy the resulting spotsh binary into a convenient directory in your $PATH (e.g. $HOME/bin)

## Usage

```bash
spotsh
```
